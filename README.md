# VirelaTech

A React + TypeScript starter built with Vite, Tailwind CSS, and Supabase.

## Getting started

### Requirements

- Node.js 18+ (or compatible LTS)
- npm (or yarn / pnpm)

### Local development

```sh
# Clone repository
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>

# Install dependencies
npm install

# Start dev server
npm run dev
```

### Environment variables

Create a `.env` file in the project root and provide Supabase keys:

```env
VITE_SUPABASE_URL=your-supabase-url
VITE_SUPABASE_PUBLISHABLE_KEY=your-supabase-key
```

## Build & preview

```sh
npm run build
npm run preview
```

## Technologies used

- Vite
- React
- TypeScript
- Tailwind CSS
- Supabase

If you're maintaining this project, keep dependencies up to date and run `npm run lint` / `npm run test` regularly.
